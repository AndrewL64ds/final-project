# Boll Deluxe Website

## Creator

Andrew Long

## Description

This project is a brand-new new website for Boll Deluxe, a fan-made Mario & Sonic game. While Boll has a itch.io page, since that's where the game download is hosted, I felt like it could have a custom website to go more in-depth about the game and its features. This site will include as much information to get people to understand what the game has and can do. However, the site will be done in a way that you can be the basics fast. The primary goal

## Target Audience

- **Primary:** Those who are interested in fan-games who want to learn as many details about Boll as they can.
  
- **Secondary:** Those who want the basic information about Boll quickly to see if it's worth their time.

## Project Goals

- Design a website that has a lot of information while being easily readable at a glance.
- Include as much information about characters, items, modes, etc.
- Highlight what players can do in the game.
